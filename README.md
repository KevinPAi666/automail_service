# automail_service
前置準備:
1. 至google服務申請"應用程式密碼"
2. 至google API服務申請"服務帳戶"
3. 至google API服務申請"私密金鑰"

使用步驟:
1. 將本專案clone到自己的電腦, 根據automail.py中的指示填入個人資訊(帳號/密碼/json名稱/表單後台連結)
2. 上傳github並設為"私密"
3. 使用render服務部署第2.點的.git路徑
4. 部屬成功後至表單後台 工具列>"擴充功能">"APP Script">填入自己的render網址
5. 完成
